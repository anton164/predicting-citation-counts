# aml-research-project


## Set-up (Python 3.7+)
1. Install dependencies
```
pip install -r requirements.txt
```
2. Copy datasets to the repository manually
   - 250k.docs.jsonl (sample of 250k docs)
   - mag5.docs.jsonl (full dataset with 5 mill docs)
